# perpe`tui`ty

The Terminal UI (TUI) library that keeps on giving.

<!-- prettier-ignore-start -->
> [!NOTE]
> This library is currently a work in progress. Until it reaches version 1.0, expect breaking changes.
<!-- prettier-ignore-end -->

`perpetuity` provides the ability to specify your TUI declaratively, similar to React, with core
language primitives. That means no build tools, a much smaller core, and fewer concepts to learn.

## Installation

```
npm add perpetuity
```
